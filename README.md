This is a blog apllication where you can input a title and write about what you want to blog about, 
there is a delete and edit function that has been added to the application in the latest update.

---------------------------Reflection---------------------------
My biggest obstacle while working on this application was that I forgot to add the defer attribute to
my Javascript tag so I spent an enormous amount of time editing and researching why my application was not
functioning correctly just to take a look at my HTML and see I forgot my defer tag.
